# Image Rules

## Active rule

All menu imagery uses a consistent **4:5 ratio**.

## Current state

The project currently uses **local placeholder artwork** under:

```text
frontend/public/media/placeholders/
```

These files are lightweight stand-ins for future production WebP images.

## Replacement rules for final WebP assets

When real images are ready later:

1. Keep the same filenames where possible.
2. Preserve the 4:5 crop.
3. Avoid mixed lighting styles across categories.
4. Keep backgrounds clean and believable.
5. Prefer restrained editorial food photography over noisy stock imagery.

## Why 4:5 stays fixed

A single image ratio keeps the catalog calmer, improves scan rhythm, and prevents the grid from feeling improvised.
