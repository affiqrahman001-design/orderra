# Content Guide

## Voice

ORDERra v2 should sound:

- calm
- premium
- natural
- operationally clear
- not over-marketed
- not AI-styled

## Preferred copy style

Use short, grounded phrases that feel like real product language.

### Good

- Freshly prepared favourites
- Kitchen-ready in around 25 to 35 minutes
- Add a quick note for the kitchen
- Delivery details
- Need help with allergies or group orders?

### Avoid

- Hyperbolic sales language
- Synthetic “future of food” phrasing
- Anything that sounds like auto-generated placeholder text
- Repetitive CTA labels across unrelated actions

## CTA guidance

Use different actions for different intents:

- `Add now`
- `Quick customise`
- `Build order`
- `Continue to checkout`
- `Place order`
- `Back to menu`

## Demo framing

The visible UI should stay believable. Explanations that this is a portfolio piece belong in docs, not in the main ordering experience.
