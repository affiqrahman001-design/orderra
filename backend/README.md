# ORDERra Backend

This is the Laravel API backend for ORDERra, a premium single-restaurant burger ordering platform demo.

The backend is designed as a demo-safe reference architecture for:

- catalog
- cart
- pricing
- orders
- fulfillment
- dine-in QR sessions
- split bill
- payment simulation
- refund simulation
- rider simulation
- webhook simulation
- support tickets
- admin reference operations
- audit logs

ORDERra backend does not process real payments, real payouts, live webhooks, or live delivery dispatch.

---

## Stack

- Laravel
- PHP 8.3+
- SQLite for local demo
- PostgreSQL-ready architecture
- REST-style API
- Queue/cache/realtime-ready structure for future upgrades

---

## Local Setup

Install dependencies:

```bash
composer install
```
